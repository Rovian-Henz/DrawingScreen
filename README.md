# Drawing Screen

Creates a drawing screen like gartic and a server to other people see the drawing


## Installation

All dependencies in NPM

```bash
npm install
```

## Usage

```bash
node app
```

Open localhost:3000 and start drawing

## Techs
node, express, socket.io
